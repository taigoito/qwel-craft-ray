# Ray Simulator | Created by ― QWEL.DESIGN

昔プレイしたゲーム「サガ・フロンティア」の作中に出て来た「光の迷宮」のロジック実装に挑戦。  
分光・集光の性質を上手く使って、最終的に白い光をゴールに導くことができればクリア。  
キーボードで操作（マウス操作にすればもっと遊び易かった・・・）。
